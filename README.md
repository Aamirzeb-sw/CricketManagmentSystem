# Cricket Management System (Java Swing)

A professional-looking Cricket Management System built using **Stack (LIFO)** data structure in **Java Swing**.

### Features:
- Add (Push) player
- Remove last player (Pop)
- Update / Delete by ID
- Search by ID or name (Live)
- Clean modern light UI

### Data Structure:
Implemented using Java **Stack** class for managing player records (LIFO order).
